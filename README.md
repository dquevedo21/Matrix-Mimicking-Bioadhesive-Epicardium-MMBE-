# Matrix-Mimicking-Bioadhesive-Epicardium-MMBE
Repository for the paper "A matrix-mimicking bioadhesive epicardium for tunable modulation of biomechanics in the acutely infarcted heart" by Claudia E. Varela, Diego A. Quevedo-Moreno, Jean Bonnemain, Keegan L. Mendez, Yiling Fan, Jonathan Tagoe, Carly Long, Mossab Saeed, Hyunwoo Yuk, Xuanhe Zhao and Ellen T. Roche. 

This paper has been submitted to Nature Materials.

Preprint: https://www.biorxiv.org/content/10.1101/2025.10.07.681040v1.abstract
